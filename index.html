Recursos

Tudo o que você precisa para ler

Feito por um leitor de quadrinhos, para leitores de quadrinhos.

📚

Biblioteca inteligente

Capas em grade carregadas em segundo plano. Filtros por status: não lidos, lendo, concluídos e favoritos.

🔍

Busca rápida

Filtre por título, série, autor ou editora. Acesso imediato sem precisar rolar a biblioteca inteira.

📖

Leitor completo

Zoom com cursor, arrastar, rotação, controle de brilho, modo imersivo e barra de progresso clicável.

▭▭

Página dupla

Visualize duas páginas lado a lado, inclusive no modo mangá — do jeito que o artista imaginou.

🖼️

Miniaturas

Faixa com todas as páginas em miniatura. Clique em qualquer uma para pular direto.

📁

Coleções automáticas

Subpastas viram coleções. Arquivos com nomes similares são agrupados em séries com progresso unificado.

🔖

Bookmarks

Marque páginas importantes. Todos os bookmarks ficam visíveis na barra de progresso.

💾

Backup e restauração

Exporte progresso, bookmarks e favoritos em JSON. Útil ao trocar de computador.

Formatos

Seus arquivos, do seu jeito

Suporte nativo a CBZ e suporte a CBR e PDF com dependências simples.

.cbz
CBZ / ZIP
Suporte nativo — nenhuma dependência
.cbr
CBR / RAR
Requer WinRAR instalado
.pdf
PDF
Requer pip install pymupdf

Instalação

Instalador ou código fonte

Execute o instalador .exe ou rode direto do Python se preferir.

cmd — instalação pelo código fonte
# Clone o repositório
> git clone https://github.com/lucrazy-fn/PANEL-ComicBookReader
> cd PANEL-ComicBookReader

# Instale as dependências
> python -m pip install pillow rarfile pymupdf

# Execute
> python ComicReader.py
